## 🛠️ 本地开发

本项目基于 [VitePress](https://vitepress.dev) 构建，即使你不是程序员也能轻松上手。

### 环境要求
- [Node.js](https://nodejs.org/) 18+
- [pnpm](https://pnpm.io/) 9+

### 1. 克隆项目
```bash
git clone [https://github.com/你的用户名/你的仓库名.git](https://github.com/你的用户名/你的仓库名.git)
cd 你的仓库名

```

### 2. 安装依赖

```bash
pnpm install

```

### 3. 启动本地预览

```bash
pnpm dev

```

访问 `http://localhost:5173` 即可实时预览修改效果。

### 4. 构建打包

```bash
pnpm build

```

## 🚢 部署 (Deployment)

推荐使用 **Cloudflare Pages** 进行自动化部署，零成本且速度极快。

1. Fork 本仓库到你的 GitHub。
2. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/) -> **Workers & Pages** -> **Create Application**。
3. 连接 GitHub，选择本项目。
4. 构建配置：
* **Framework preset**: `VitePress`
* **Build command**: `pnpm build`
* **Build output directory**: `.vitepress/dist`


5. 点击 Deploy，完成！
